# 16Personalities 자동화 — 설치 안내

## 사전 준비 (최초 1회)

### 1. Node.js 설치
https://nodejs.org 에서 LTS 버전 다운로드 후 설치

설치 확인:
```
node --version   # v20 이상
npm --version
```

### 2. 프로젝트 파일 설치
이 폴더를 바탕화면에 복사한 후 터미널에서:
```
cd ~/Desktop/인적성검사\ 자동화
npm install
npm run install:browsers
```

### 3. 인증 파일 설정
별도로 전달받은 `service-account-key.json` 파일을
이 폴더 안에 복사

### 4. 환경 설정
```
cp .env.example .env
```
.env 파일은 수정 불필요 (기본값 사용)

---

## 서버 실행 방법

### 시작
```
cd ~/Desktop/인적성검사\ 자동화
npm run dev
```
아래 메시지가 뜨면 준비 완료:
```
Pub/Sub 구독 시작: form-submissions-mac
```

### 종료
터미널에서 Ctrl + C

---

## 사용 방법

1. 서버 실행 (위 방법으로)
2. 팀원이 구글폼 제출
3. 터미널에서 처리 진행 로그 확인
4. 2~5분 후 구글 시트에 결과 자동 기록
5. 작업 완료 후 서버 종료 (Ctrl+C)

---

## 주의사항

- 서버가 실행 중일 때만 자동 처리됩니다
- 서버가 꺼져 있어도 메시지는 큐에 보관됩니다
  (서버 켜면 밀린 것들이 순서대로 처리됨)
- 여러 명이 동시에 서버를 켜도 중복 처리되지 않습니다
