# 📑 월급쟁이부자들 인적성검사 자동화 운영 매뉴얼

안녕하세요 시아님! 이 시스템을 매일매일 똑똑하게 운영하기 위한 핵심 가이드입니다. ✨
혹시 저와 대화한 내용이 기억나지 않으실 때 이 파일을 열어보세요!

---

## 🏃‍♂️ 1. 매일 아침 출근해서 실행하기
아침에 맥북을 켜시면 가장 먼저 터미널에서 이 명령어를 입력해 주세요. (자동화 로봇을 깨우는 일입니다! 🤖)

1. **터미널을 엽니다.** (Command + Space → '터미널' 검색)
2. **아래 명령어를 그대로 복사해서 엔터!**
   ```bash
   cd ~/Desktop/인적성검사\ 자동화
   npm run dev
   ```
3. **`✅ Pub/Sub 구독 시작: form-submissions-mac`** 이라는 초록색 체크 메시지가 뜨면 완료! 이제 자동화 대기 상태입니다.

---

## 🔗 2. 설문 사이트 주소 (지원자에게 보내는 링크)
시아님이 직접 만든 예쁜 설문 사이트 주소예요! 이 주소를 지원자에게 보내주시면 됩니다.

*   **인터넷 주소 (GitHub Pages)**: `https://sia-debug.github.io/-/survey-app/index.html`
*   **사용법**: 태블릿이나 폰, PC 어디서든 이 주소를 치면 설문이 열립니다.

---

## 💻 3. 코드를 수정했을 때 (깃허브에 올리기)
설문지 디자인이나 문구를 고치고 나서 깃허브에 최신판을 올리고 싶을 때 사용하세요.

1. **터미널에서 `Control` + `C` 를 눌러 실행 중인 서버를 잠시 멈춥니다.**
2. **아래 명령어를 순서대로 한 줄씩 입력하세요.**
   ```bash
   git add .
   git commit -m "최신 설문 수정 완료"
   git push
   ```
3. **완료 후 다시 `npm run dev` 를 쳐서 서버를 켭니다!**

---

## 🔔 4. 슬랙(Slack) 알림 연동법
분석이 끝나자마자 슬랙으로 결과를 받고 싶을 때 사용하는 방법입니다.

1. **슬랙 앱에서 'Incoming Webhooks'를 생성하고 URL을 복사합니다.**
2. **폴더 내의 `.env` 파일을 엽니다.**
3. **`SLACK_WEBHOOK_URL=` 옆에 복사한 주소를 붙여넣고 저장합니다.**
4. **터미널을 껐다가 다시 켜면(npm run dev) 알림이 활성화됩니다!**

---

## 🛡️ 5. 보안 및 주의사항
*   **service-account-key.json**: 인적성 로봇의 '열쇠'입니다. 절대 삭제하거나 외부에 공유하지 마세요!
*   **맥 유지**: 자동화 로봇은 시아님의 맥이 켜져 있을 때만 작동합니다. (제출된 설문은 서버에 쌓였다가 맥을 켜는 순간 한꺼번에 처리되니 안심하세요!)
